## Spacing

Spacing between columns is set using the `space` prop. By default, `space` is `0`.

## Column Widths

Column widths are set manually using the `span` prop. The total `span` amount of each column can be of different values but should not exceed the `count`.

## Collapsing Columns

Columns can be collapsed into a single column when the viewport is below a certain width determined by the optional prop `collapseBreakpoint`. The columns below collapse when the viewport is less than or equal to 768px.

<Canvas of={ColumnsStories.Collapsing} />
